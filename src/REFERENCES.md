## API Coverage
- **SOQL**: query records, query all records (including deleted/archived)
- **Objects**: list objects, describe object schema and fields
- **Records CRUD**: get, create, update, delete, upsert records on any standard or custom object
- **Leads**: list leads, create lead, convert lead to contact/account/opportunity
- **Contacts**: list contacts, create contact, update contact
- **Opportunities**: list opportunities, create opportunity, update opportunity
- **Accounts**: list accounts, create account, update account
- **Cases**: list cases, create case, update case, close case
- **Reports**: list reports, run report with runtime filters, get report metadata
- **Users**: list users, get user details
- **Bulk**: bulk insert, bulk update, bulk delete (async job-based)
- **Metadata**: list metadata types, deploy metadata zip packages with validation
